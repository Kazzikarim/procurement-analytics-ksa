# procurement-analytics-ksa

Title: Procurement Analytics Dashboard (KSA Oil & Gas Simulation)

Business problem:
Procurement teams need visibility into cycle time, supplier performance, and spend leakage to reduce delays and control cost.

KPIs included:

PR to PO cycle time

On-time delivery %

Price variance % vs contract

Spend by category & department

Supplier concentration risk

Tools: Power BI, Excel, SQL (optional)

Files:

/data: CSV tables

/powerbi: PBIX file

/screenshots: dashboard images

/sql: queries for KPI calculations
